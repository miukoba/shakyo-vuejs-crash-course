# shakyo-vuejs-crash-course

https://www.youtube.com/watch?v=qZXt1Aom3Cs の写経

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
# frontend
yarn run serve

# backend
yarn run backend
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
